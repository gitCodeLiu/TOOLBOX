# 文档目录

本目录存放项目各模块的详细文档。根目录 `README.md` 中仅保留摘要和链接，完整说明见此处。

## 工具文档（docs/tools/）

| 工具名称 | 文档文件 | 对应路径 |

## 维护规则

- **新增工具**：在 `docs/tools/` 下创建 `<tool-id>.md`，并在上方工具文档表格补充一行
- **文档命名**：与工具 ID 保持一致（即 `src/tools/<tool-id>/` 中的目录名）
- **内容要求**：工具文档需包含功能列表、数据格式说明、已知限制、后续优化方向
