# 单位换算

**工具路径**：`/tools/unit-converter`  
**工具分类**：实用工具  
**文件位置**：`src/tools/unit-converter/UnitConverterPage.tsx`

---

## 支持的换算类别

| 类别 | 单位示例 |
|------|---------|
| 长度 | mm、cm、m、km、英寸、英尺、码、英里、海里 |
| 重量 | mg、g、kg、吨、磅、盎司、斤 |
| 面积 | mm²、cm²、m²、km²、公顷、英亩、亩 |
| 体积 | mL、L、m³、杯、品脱、加仑 |
| 温度 | 摄氏度、华氏度、开尔文 |
| 数据存储 | bit、B、KB、MB、GB、TB |
| 速度 | m/s、km/h、mph、节 |

## 功能特点

- **分类标签**：点击顶部分类切换换算类别
- **实时结果**：选择源单位和目标单位后实时显示结果
- **全量展示**：底部网格展示该类别所有单位的换算结果
- **科学计数法**：极大或极小数值自动切换为科学计数法显示

## 已知限制

- 温度换算不基于乘法因子，而是通过公式转换，属特殊处理

## 后续优化方向

- 支持货币汇率换算（需接入实时汇率 API）
- 支持更多工程单位（压力、功率、能量等）
