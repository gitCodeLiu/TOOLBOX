# 时间戳转换

**工具路径**：`/tools/timestamp-converter`  
**工具分类**：实用工具  
**文件位置**：`src/tools/timestamp-converter/TimestampConverterPage.tsx`

---

## 功能列表

- **实时时间戳显示**：顶部显示当前秒级 Unix 时间戳，每秒自动刷新
- **时间戳 → 日期**：输入 Unix 时间戳，转换为人类可读的日期时间
- **日期 → 时间戳**：输入日期时间字符串，转换为 Unix 时间戳
- **单位切换**：支持秒（s）和毫秒（ms）两种时间戳格式
- **时区选择**：支持 UTC、北京、东京、纽约、洛杉矶等常用时区
- **多时区对照**：同时展示同一时间点在 8 个时区的对应时间

## 支持的时区

UTC+0、UTC+8（北京）、UTC+9（东京）、UTC+5:30（印度）、UTC-5（纽约）、UTC-8（洛杉矶）、UTC+1（伦敦）、UTC+2（巴黎）

## 已知限制

- 仅支持正整数时间戳（> 0），即 1970 年以后的时间

## 后续优化方向

- 支持更多时区
- 支持 ISO 8601 格式输入
