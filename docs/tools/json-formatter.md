# JSON 格式化

**工具路径**：`/tools/json-formatter`  
**工具分类**：格式转换  
**文件位置**：`src/tools/json-formatter/JsonFormatterPage.tsx`

---

## 功能列表

- **格式化**：将压缩或混乱的 JSON 按指定缩进（2空格/4空格/Tab）进行美化输出
- **压缩（Minify）**：将 JSON 压缩为单行，减小体积
- **校验**：检测 JSON 是否合法，显示错误信息
- **键名排序**：可选对 JSON 对象的键名按字母顺序排序
- **一键复制**：复制输出结果至剪贴板

## 操作说明

1. 在左侧输入框粘贴或输入 JSON 字符串
2. 选择缩进方式（默认 2 空格）
3. 点击「格式化」按钮美化，点击「压缩」按钮压缩
4. 「校验」按钮仅验证合法性，不修改输出
5. 右侧显示结果，可点击「复制」按钮复制

## 已知限制

- 仅支持标准 JSON，不支持 JSON5（含注释、尾随逗号的 JSON）
- 超大 JSON（>10MB）可能导致页面卡顿
