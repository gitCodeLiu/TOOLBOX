# 📈 Excel 转折线图

**路径**：`/tools/excel-to-chart`

将 Excel 数据文件转换为可交互的折线图，支持多系列对比、数据处理、图表导出等完整功能。

---

## Excel 格式约定

- **第 1 行**（表头）：第一列是 X 轴列名（可任意填写），其余列是各 Y 轴系列名称
- **第 2 行起**（数据）：第一列是 X 轴标签值，其余列填写对应系列的数值

**示例**：

| 日期（X轴） | 销售额（Y轴系列一） | 利润（Y轴系列二） |
|:-----------:|:------------------:|:----------------:|
| 1月         | 120                | 30               |
| 2月         | 150                | 45               |
| 3月         | 130                | 38               |
| 4月         | 180                | 62               |

## 功能列表

### 文件上传

| 功能 | 说明 |
|------|------|
| 拖拽上传 | 将文件拖入上传区域即可解析 |
| 点击上传 | 点击区域弹出文件选择对话框 |
| 格式支持 | 兼容 `.xlsx`、`.xls`、`.csv` |

### 图表展示

| 功能 | 说明 |
|------|------|
| 多系列支持 | 每列数据作为独立折线系列 |
| 显示/隐藏折线 | 点击系列控制面板的眼睛图标切换可见性（最少保留一条） |
| 自定义颜色 | 点击色块自由修改每条折线的颜色 |
| 图表类型切换 | 支持折线图和面积图两种模式 |
| 曲线平滑 | 切换折线为光滑曲线或折线段 |
| 数据点显示 | 控制是否在折线上展示每个数据点 |
| 网格控制 | 可显示/隐藏背景网格 |

### 数据处理

数据处理按以下固定顺序执行，每步可独立开关：

| 步骤 | 功能 | 说明 |
|------|------|------|
| 步骤 1 | X 轴聚合 | 将相同 X 轴标签的行合并，重复项数值求和 |
| 步骤 2 | X 轴排序 | 按 X 轴标签升序排列（支持数字前缀、月份等格式） |
| 步骤 3 | 累加求和 | 每个数据点 = 自身值 + 前面所有点之和（Running Total） |

**数据处理示例**：

原始数据（含重复 1 月）：

| 日期 | 销售额 |
|------|--------|
| 1月  | 120    |
| 2月  | 150    |
| 3月  | 130    |
| 1月  | 150    |

经过聚合 → 排序 → 累加后：

| 日期 | 销售额 |
|------|--------|
| 1月  | 270    |（120+150，聚合）
| 2月  | 420    |（270+150，累加）
| 3月  | 550    |（420+130，累加）

### 导出与复制

| 功能 | 说明 |
|------|------|
| 导出 PNG | 将当前图表导出为 2x 高清 PNG 图片 |
| 导出 SVG | 将当前图表导出为矢量 SVG 文件 |
| 复制数据 | 将原始数据以 Tab 分隔格式复制到剪贴板（可直接粘贴回 Excel） |
| 数据预览 | 在图表下方展示原始数据表格，支持横向和纵向滚动 |

## 已知限制

| 问题 | 说明 |
|------|------|
| PNG 导出质量 | 使用 Canvas 2x 缩放，复杂图表可能有轻微锯齿 |
| 大文件性能 | Excel 行数超过 10000 行时前端解析可能较慢 |
| 多 Sheet | 目前仅解析第一个 Sheet |
