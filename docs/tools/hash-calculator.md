# 哈希计算

**工具路径**：`/tools/hash-calculator`  
**工具分类**：编码/加密  
**文件位置**：`src/tools/hash-calculator/HashCalculatorPage.tsx`

---

## 功能列表

- **实时计算**：输入文本即时显示所有哈希结果，无需点击按钮
- **支持算法**：MD5、SHA-1、SHA-256、SHA-512、SHA-3 (256位)、RIPEMD-160
- **大小写切换**：支持小写（默认）和大写两种输出格式
- **一键复制**：每条哈希值旁均有复制按钮

## 依赖

- `crypto-js` 库（支持 MD5、SHA 系列、RIPEMD-160）

## 已知限制

- 仅支持字符串哈希，不支持文件哈希
- 超长文本（>1MB）可能有轻微延迟

## 后续优化方向

- 支持文件拖拽上传并计算文件哈希
- 支持 HMAC 消息认证码
