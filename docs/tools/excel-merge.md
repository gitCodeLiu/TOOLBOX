# Excel 关联合并

## 功能概述

上传**源表（主表）**与**增强表**，通过 **1 个或 2 个**关联字段将两行数据对应起来。以源表的行数与列顺序为基准，把增强表中匹配到的行里的列**补充**到右侧，或按选项**覆盖**同名列，最后导出为新的 `.xlsx` 文件。全部在浏览器本地完成，数据不上传服务器。

## 使用步骤

1. 分别上传源表与增强表（支持 `.xlsx`、`.xls`、`.csv`）。
2. 选择「单字段关联」或「双字段关联」，并在两侧下拉框中选定用于匹配的列名（两端列名可以不同，只要语义对应即可）。
3. 选择同名列冲突时的策略：保留源表值，或用增强表覆盖。
4. 点击「合并并下载 Excel」生成文件。

## 数据约定

- 默认读取两个文件中的**第一个工作表**。
- **第一行**为表头；空表头会自动命名为 `列1`、`列2`…，重名列会自动加后缀区分。
- 关联字段按单元格文本比较（首尾去空格）；空单元格参与匹配时视为空字符串。

## 合并规则摘要

| 场景 | 行为 |
|------|------|
| 输出行数 | 与源表一致 |
| 增强表独有的列 | 追加在源表所有列的右侧 |
| 源表某行未在增强表中找到匹配键 | 增强表独有列填空，源表原有列不变 |
| 增强表中多条行具有相同关联键 | 以**最后一条**为准，并提示重复键条数 |
| 两表同名列（且该列在增强表中不是关联键） | 按「保留源表 / 用增强表覆盖」选项处理 |
| 增强表中作为关联键使用的列 | 不会作为新列重复追加 |

## 已知限制

- 不支持多工作表批量合并；需分次导出或事先拆到首表。
- 超大文件可能导致浏览器内存占用较高，建议单表行数在合理范围内使用。

## 后续可优化方向

- 支持选择工作表、导出为 CSV。
- 未匹配行高亮或单独导出。
- 关联前字段类型规范化（如日期、数字格式统一）。
